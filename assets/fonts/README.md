# Brobane font

Place **Brobane.ttf** (or Brobane.otf) here for the app bar page titles.

Download from [DaFont](https://www.dafont.com/brobane.font) or [BeFonts](https://befonts.com/brobane-font.html).  
If the file is named differently (e.g. `Brobane-Regular.ttf`), rename it to `Brobane.ttf` or update `pubspec.yaml` under `flutter.fonts` to match the filename.
